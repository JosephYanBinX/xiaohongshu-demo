# Skills 激励体系设计（精简版）

Skills 不适合一口价买卖——用户在使用前无法判断一个 Skill 是否适合自己，而优质 Skill 的价值也会随使用深度持续增长。更合适的方式是平台统一货币化，创作者按贡献分成。

**用户侧**，通过「点点 AI Pro 会员」解锁所有 Skills 无限用量，免费用户享有每月体验额度。单个 Skill 不单独定价，降低用户的决策门槛。

**创作者侧**，平台将会员收入的固定比例注入创作者收益池，按月根据安装量、活跃使用次数、用户留存率和 Skill 评分综合分配。创作者无需定价、无需处理客服，专注把 Skill 做好即可。

**推广侧**，复用小红书现有的内容带货逻辑：Skill 笔记获得额外流量加权，创作者可查看每篇笔记带来的安装转化；任何用户都可以生成推广链接，他人通过链接安装并使用后，推广者获得佣金分成。这套机制对小红书用户来说并不陌生。

**冷启动阶段**，平台全额补贴，所有 Skills 免费使用，创作者激励直接由平台发放。目的是先把供给侧做起来——创作者有了现金激励，愿意认真做 Skill、发笔记推广；内容供给充足后，用户使用习惯自然形成，后续付费转化才有基础。
